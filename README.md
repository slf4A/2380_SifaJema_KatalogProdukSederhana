# Tugas 3 Website Katalog Produk Django
Sifa Jema (2308107010080)

## Penjelasan singkat terkait Program Katalog
Program ini merupakan website katalog produk sederhana menggunakan framework Django.

Website memiliki beberapa halaman:
- Homepage
- Daftar Produk
- Detail Produk
- Halaman Kontak

Data produk ditampilkan menggunakan data hardcoded tanpa database.
